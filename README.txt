O Vírus SkinWalker foi feito em JavaScript, VBScript e Bat. E compilado para .exe

Ele apaga alguns dados do registro do windows, Apaga a pasta Windows e exibe vários programas para o usuário não consegui desativar o executavel.

O Vírus SkinWalker foi feito pelo Renato
Discord: Renatin#2004


Observações
	1. Eu não me responsabilizo pelas suas atividades com o malware. Isso pode causar na destruição total do seu computador. O Vírus apaga a pasta Windows, Program Files e Program Files (x86). Então o Windows não irá mais bootar caso o mesmo não recuperar o dvd ou pendrive com a iso do windows instalada

	2. Esse vírus foi feito a pedido do pessoal do BPD (Bots para Discord) com um evento de halloween que eles mesmo fizeram. O administradores pediram para fazer uma aplicação em referência ao Halloween. E aqui está!
